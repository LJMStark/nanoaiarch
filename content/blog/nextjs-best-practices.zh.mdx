AI 建筑可视化正在改变设计师的工作方式。本文汇集了专业建筑师在使用 Arch AI 时总结的最佳实践。

## 素材准备

### 草图的黄金法则

好的草图是获得优质效果图的基础：

**线条处理**
- 使用清晰的黑色线条
- 主要结构线条加粗
- 辅助线条保持细腻
- 避免过多涂改痕迹

**构图建议**
- 选择最能表达设计意图的视角
- 保持适当的留白
- 重要元素居中或遵循三分法
- 考虑最终效果图的画幅比例

**标注技巧**
- 简单的材质标注有助于 AI 理解
- 可以标注大致的尺寸比例
- 用文字说明特殊的设计意图

### 白模的最佳实践

从 3D 软件导出白模时：

| 软件 | 推荐设置 | 注意事项 |
|------|----------|----------|
| SketchUp | 风格设为"隐藏线" | 关闭阴影获得更干净的轮廓 |
| Rhino | 使用"Arctic"显示模式 | 确保曲面完整无破面 |
| Revit | 隐藏所有标注和标签 | 选择合适的详细程度 |
| Blender | Eevee渲染白色材质 | 添加简单的环境光 |

**视角选择**
- 人视点高度约 1.6m
- 保持适当的透视感
- 避免极端的广角或长焦效果
- 确保主要建筑元素完整入画

## 提示词技巧

### 结构化描述

按照以下顺序组织你的提示词：

```
1. 建筑类型和规模
2. 建筑风格
3. 主要材质
4. 环境和场景
5. 光线和氛围
6. 摄影风格
```

**完整示例**

```
两层现代住宅，
极简主义风格，清水混凝土外墙搭配大面积玻璃幕墙，
位于山坡上，周围是针叶林，
傍晚时分的金色阳光从侧面照入，
专业建筑摄影，广角镜头，高清晰度
```

## 工作流程

### 方案探索阶段

快速生成多个方向：

1. 使用 1K 画质节省时间
2. 每次尝试不同的风格方向
3. 保存有潜力的版本
4. 收集客户反馈

### 方案深化阶段

确定方向后深入细节：

1. 切换到 2K 画质
2. 在对话中逐步优化细节
3. 尝试不同的材质搭配
4. 调整光线和氛围

### 正式出图阶段

最终效果的打磨：

1. 使用 4K 画质
2. 确保每个细节都符合预期
3. 生成多个视角
4. 导出高分辨率文件

## 常见场景处理

### 室内空间

室内效果图的特殊技巧：

- 明确描述空间功能（客厅、卧室、办公室）
- 指定主要的家具风格
- 注意描述光源位置（窗户、灯具）
- 考虑软装配饰的风格

```
现代简约风格的开放式客厅，
3.5米层高，落地窗朝向花园，
灰色布艺沙发搭配胡桃木茶几，
午后阳光从窗户洒入，温馨舒适的氛围
```

### 夜景效果

夜景渲染的要点：

- 强调建筑的灯光效果
- 描述室内透出的光线
- 考虑景观照明
- 注意天空的处理

```
现代别墅夜景，
室内暖黄色灯光透过玻璃幕墙，
庭院有地埋灯勾勒出景观轮廓，
深蓝色的夜空，远处城市灯火隐约可见
```

### 鸟瞰效果

鸟瞰图的注意事项：

- 说明视角高度和角度
- 描述周边环境
- 注意建筑群的整体关系
- 考虑景观和道路

```
住宅区鸟瞰效果图，
45度俯视角度，高度约100米，
12栋错落布置的现代住宅楼，
中央有景观公园和人工湖，
周边是城市道路和商业配套
```

## 质量把控

### 检查清单

生成效果图后确认：

- [ ] 建筑比例是否正确
- [ ] 材质表现是否符合预期
- [ ] 光影关系是否合理
- [ ] 环境元素是否协调
- [ ] 整体氛围是否到位

### 常见问题修复

| 问题 | 解决方案 |
|------|----------|
| 比例失真 | 在提示词中明确建筑尺度 |
| 材质不对 | 使用更具体的材质描述词 |
| 光线怪异 | 指定具体的时间和方向 |
| 细节模糊 | 提升画质或分步生成 |
