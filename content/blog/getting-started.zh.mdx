不要从零开始。MKSaaS 提供了启动 SaaS 产品所需的一切，让你用几天而不是几个月来完成。

## 为什么选择 MKSaaS？

每个 SaaS 产品都需要相同的基础：用户认证、支付处理、数据库管理、国际化和现代 UI。从零构建这些需要数周时间。MKSaaS 为你提供一个可定制的生产级模板。

## 内置功能

| 功能 | 说明 |
|------|------|
| **认证** | 邮箱/密码 + 社交登录（Google、GitHub） |
| **支付** | Stripe 集成，支持订阅和一次性购买 |
| **数据库** | PostgreSQL + Drizzle ORM |
| **AI 功能** | 多提供商图像生成 |
| **国际化** | 开箱即用的中英文支持 |
| **UI 组件** | Radix UI + TailwindCSS |

## 快速开始

克隆并安装：

```bash
git clone https://github.com/mksaas/mksaas-template.git
cd mksaas-template
pnpm install
```

配置环境变量：

```bash
cp env.example .env
# 编辑 .env 填入你的 API 密钥
```

启动开发服务器：

```bash
pnpm dev
```

访问 `http://localhost:3000`，开始构建。

## 项目结构

```
src/
├── app/           # Next.js App Router 页面
├── components/    # 可复用 React 组件
├── lib/           # 工具函数和辅助函数
├── db/            # 数据库 schema 和迁移
├── actions/       # 服务端操作
└── config/        # 应用配置
```

## 下一步

1. **自定义设计** - 在 `src/config/website.tsx` 中更新颜色和品牌
2. **配置认证** - 在 Better Auth 中设置 OAuth 提供商
3. **添加 Stripe 密钥** - 在仪表板中启用支付
4. **部署到 Vercel** - 一键部署
