## Usage Tips

### Maximize Conversational Editing

1. **Establish Direction First** - Use first generation to set overall style
2. **Step-by-Step Modifications** - Change only one or two elements at a time
3. **Leverage Context** - AI remembers previous conversation content
4. **Be Specific** - Use concrete position descriptions like "the window on the left"

### Efficient Project Management

1. **Separate by Scheme** - Manage different design concepts in separate projects
2. **Name Promptly** - Give projects meaningful names
3. **Mark Important Versions** - Favorite satisfactory renderings
4. **Regular Cleanup** - Delete unneeded test images
